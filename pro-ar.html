AR Shopping
AR Shopping is a project that uses Augmented Reality to improve online shopping experience. Users can see virtual products in their real environment by this app, thus having a more intuitive sense about the product.
Problems of online shopping
Take chair as an example:
  • How big is this chair?
  • Does it match the table in my room?
Take tie as an example:
  • Which color or pattern suits me best?
  • Which size fits me best?
What if users can see products in their real environment before paying for it online?
Here are some of my AR products, users can see AR models of necklace, hat, dress, desk etc.
Our advantages
  • Auto 3D Modelling and easy 2D Modelling;
  • Virtual product is the same size as real product.
Modelling
3D Model
· Generate 3D AR model by multiple 2D pictures of the product from 360° angles using 123D Catch software.
2D Model
· Simply need a 2D picture of the product to make an AR model.
Design
1. Desktop: link eBay website to AR shopping app by QR code.
2. Mobile: can be used independently from desktop.
AR Demo -- Tie
Users can try on any types of tie by clicking different buttons, so that they can have an intuitive sense about how the tie suits them.
Made with Unity and Eclipse
AR Demo -- Folding Lamp
For folding lamp with different sizes and colors, users can see the folding animation by clicking "Play" button, or see it in different sizes and colors by clicking the buttons at the bottom.
Made with Unity and Eclipse
