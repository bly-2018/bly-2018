Quantitative Research For Insightful Operations (QRIO)
QRIO is a self-service buyer dashboard creating tool where eBay analysts, PMs and managers can create their own dashboard about certain group of eBay buyer.
Demand Analysis
QRIO targets users such as eBay data analysts, PMs, managers, etc. They want to have a better self-service experience where they can create their own user cohorts and segments, and analyze the data based on the cohorts or segments.
User flow
To create a dashboard, users need to follow the main flow, that is, create a new cohort first, then create a dashboard based on the cohort. Users can also create segmentations of the cohort if they want.
Competitive Research

Zoomdata

Zoomdata provides users with an intuitive chart creating way and guides users from step to step by choosing a chart type first then fill in with data.

https://www.zoomdata.com/docs/2.5/using-the-chart-style-tool.html

Tableau

Tableau is a highly configurable data visualization tool. All metrics and dimensions can be self-defined and used in different conditions. But too much freedom sometimes confuses user.

https://www.youtube.com/watch?v=KNx_rXXLQRo&t=439s

Interana

Interana is easy to use for non-professional users. Users can add new charts and compare different charts without any tutorial.

https://demo.interana.com/?explorer
